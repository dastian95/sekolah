🚀 QUICK START:
════════════════════════════════════════════════════════════════════════════

For Students:
1. Go to: http://sekolah.local/login
2. Toggle = Siswa (default)
3. NISN: 10-digit number (e.g., 1234567890)
4. Password: Your password
5. Click "Login Siswa"
→ Redirects to /student/dashboard

For Admin:
1. Go to: http://sekolah.local/login
2. Click toggle "Admin"
3. Email: admin@sekolah.id (or username)
4. Password: Your password
5. Click "Login Admin"
→ Redirects to /admin/students

For Everyone:
- Logout: Click dropdown → Click "Logout"
- Remember me: Check "Ingat saya" for longer session

   - Never store plaintext

✅ Session Management
   - Laravel session handling
   - Encrypted session storage
   - Token regeneration post-login

✅ Guard Isolation
   - 'web' guard for admin
   - 'students' guard for students
   - Separate session storage

✅ Input Validation
   - Server-side validation
   - NISN: 10 digits
   - Email/Username detection
   - Password: 6+ characters

✅ SQL Injection Prevention
   - Eloquent ORM (parameterized queries)

✅ Error Handling
   - Generic messages (no data leaks)
   - No sensitive info exposed

✅ Token Security
   - CSRF token per form
   - Token regeneration on login
   - Session invalidation on logout
