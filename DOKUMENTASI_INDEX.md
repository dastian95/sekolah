## ✅ BEFORE YOU START

### Prerequisites

```bash
✅ PHP 8+
✅ MySQL 5.7+
✅ Composer
✅ Laravel 11
✅ Git/GitHub
```

### First Steps

```bash
1️⃣  php artisan migrate          # Run migrations
2️⃣  php artisan cache:clear      # Clear cache
3️⃣  php artisan serve            # Start server
4️⃣  http://localhost:8000        # Open in browser
5️⃣  /student-register            # Test registration
```

## 🚨 COMMON ISSUES & SOLUTIONS

### "Migration failed"

**Solution**: Check database connection in `.env`

- Database exists
- Credentials correct
- Connection active

### "Login tidak bekerja"

**Solution**: Check password hashing

- NISN must be 10 digits
- Password case-sensitive
- Session enabled

### "WhatsApp messages not sent"

**Solution**: Setup Twilio

- Read: PANDUAN_WHATSAPP_INTEGRATION.md
- Get API credentials
- Configure .env
